Arduino: Robot Siguelíneas

Completado Arduino Uno Sensor QTR-8A Driver L298N

Introducción

En este tutorial explicaremos el procedimiento para crear un robot móvil con Arduino usando nuestros conocimientos sobre el sensor de infrarrojos QTR-8A y los motores de corriente continua.

Objetivo del Proyecto: Crear un robot móvil que siga el recorrido marcado por una línea negra. El robot detectará con sensores de infrarrojos la presencia de la línea negra para rectificar su trayectoria.

Material Necesario

Componente Imagen Descripción
Plataforma Robótica Incluye:
  • 2 motores DC con ruedas
  • Rueda loca central
  • Portapilas 4xAA
  • Base de contrachapado
Arduino Uno Microcontrolador principal del proyecto
Protoboard Placa de conexiones para simplificar el cableado
Driver L298N Controlador de motores DC con control de velocidad PWM
Sensor QTR-8A Array de sensores infrarrojos para detección de línea
Cables Cables jumper para conexiones

Circuito Eléctrico

Combinamos la circuitería del driver L298N para controlar 2 motores DC y el sensor QTR-8A para detectar la línea negra.

Lógica del Algoritmo

El robot funciona según la siguiente lógica de control:

  • ✅ Ambos sensores sobre la línea: Avanzar recto
  • ↪️ Sensor izquierdo fuera: Girar a la derecha
  • ↩️ Sensor derecho fuera: Girar a la izquierda
  • ⛔ Ambos sensores fuera: Detenerse

Código Arduino

Versión 1: Con Control de Velocidad PWM

Esta versión utiliza los pines ENA y ENB del driver L298N para controlar la velocidad de los motores mediante PWM.

robot_siguelineas_pwm.ino 
/*
  Robot Siguelíneas - PPB TECH
  Control con velocidad PWM
*/

// Pines de sensores
#define SENSOR_IZQ 2
#define SENSOR_DER 3

// Pines del Driver L298N
#define ENA 5   // PWM Motor Izquierdo
#define IN1 6
#define IN2 7
#define IN3 8
#define IN4 9
#define ENB 10  // PWM Motor Derecho

int velocidad = 150;  // Velocidad (0-255)

void setup() {
  pinMode(SENSOR_IZQ, INPUT);
  pinMode(SENSOR_DER, INPUT);
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(IN3, OUTPUT);
  pinMode(IN4, OUTPUT);
  pinMode(ENB, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int sensorIzq = digitalRead(SENSOR_IZQ);
  int sensorDer = digitalRead(SENSOR_DER);
  
  if (sensorIzq == LOW && sensorDer == LOW) {
    avanzar();
  }
  else if (sensorIzq == HIGH && sensorDer == LOW) {
    girarDerecha();
  }
  else if (sensorIzq == LOW && sensorDer == HIGH) {
    girarIzquierda();
  }
  else {
    detener();
  }
}

void avanzar() {
  analogWrite(ENA, velocidad);
  analogWrite(ENB, velocidad);
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, HIGH);
  digitalWrite(IN4, LOW);
}

void girarDerecha() {
  analogWrite(ENA, velocidad);
  analogWrite(ENB, velocidad/2);
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW);
  digitalWrite(IN4, LOW);
}

void girarIzquierda() {
  analogWrite(ENA, velocidad/2);
  analogWrite(ENB, velocidad);
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, HIGH);
  digitalWrite(IN4, LOW);
}

void detener() {
  analogWrite(ENA, 0);
  analogWrite(ENB, 0);
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW);
  digitalWrite(IN4, LOW);
}

Sistema Integrado de Gestión Ambiental

Proyecto CONACYT 2025 Robot Bombero Robot Reforestador Eco Watch Station Sustentable

"Restauramos la tierra con tecnología eficiente, mitigando el cambio climático para un futuro sostenible."

Presentación del Proyecto

Somos PPBTECH, el equipo de robótica del Colegio Nacional Pablo Patricio Bogarín, ubicado en la ciudad de Ñemby. Nos especializamos en el desarrollo de proyectos innovadores con un enfoque en el cuidado del medio ambiente y el bienestar de las personas.

Este proyecto surge de la necesidad de responder a una de las problemáticas ambientales más graves que afectan a Paraguay: los incendios forestales y la deforestación, especialmente en la región del Chaco Paraguayo.

Objetivo Principal: Integrar soluciones tecnológicas a través del diseño, construcción y programación de un sistema robótico compuesto por un robot bombero, un robot reforestador y una estación de monitoreo ambiental interconectada, con el propósito de contribuir a la prevención de incendios forestales, la restauración de áreas deforestadas y la recolección de datos ambientales en Paraguay.

Problemática

El medio ambiente enfrenta hoy en día un conjunto de problemáticas que ponen en riesgo la estabilidad de los ecosistemas y la calidad de vida de las personas. En Paraguay, uno de los problemas más graves son los incendios forestales, que se producen con mayor frecuencia en épocas de sequía y que afectan principalmente al Chaco paraguayo.

Impactos Principales:

  • 🔥 Destrucción de miles de hectáreas de bosques
  • 🦜 Muerte de especies animales y pérdida de biodiversidad
  • 🌾 Pérdida de cultivos y recursos naturales
  • 💨 Emisión masiva de CO₂ y contaminación del aire
  • 🏥 Graves problemas de salud para comunidades cercanas
  • 🌳 Deforestación acelerada por expansión agrícola y ganadera

Según el INFONA, solo en enero de 2025 se registraron más de 10.000 hectáreas afectadas y 152 focos activos de incendios, lo que evidencia la magnitud del problema y la urgencia de contar con herramientas de detección y respuesta más eficaces.

Solución Integrada

La solución planteada consiste en el diseño, construcción e integración de tres prototipos robóticos complementarios que forman un sistema innovador orientado a la prevención, mitigación y restauración de los efectos de los incendios forestales.

🚒

Robot Bombero

Prototipo capaz de detectar y actuar frente a pequeños focos de incendio. Equipado con sensores de fuego y un sistema de bombeo de agua controlado por relé. Recibe señales desde la estación de monitoreo y ejecuta acciones automáticas de extinción.

  • ✓ Sensores de fuego
  • ✓ Sistema de bombeo controlado
  • ✓ Movilidad autónoma
  • ✓ Respuesta automática
🌱

Robot Reforestador

Dispositivo diseñado para transportar y depositar semillas en el suelo mediante mecanismos controlados por servomotores. Facilita el proceso de reforestación en áreas donde la regeneración natural no es suficiente.

  • ✓ Mecanismo de siembra automatizado
  • ✓ Sistema de riego inicial
  • ✓ Control por servomotores
  • ✓ Aumenta tasa de germinación
📡

Eco Watch Station

Núcleo del sistema que recopila información en tiempo real sobre temperatura, humedad y calidad del aire. Detecta humo y envía señales de alerta al robot bombero, estableciendo comunicación directa entre prototipos.

  • ✓ Sensores DHT22, MQ-2
  • ✓ Monitoreo en tiempo real
  • ✓ Detección temprana de incendios
  • ✓ Comunicación Bluetooth

Funcionamiento Técnico

El sistema sigue un ciclo técnico integrado que abarca las tres fases esenciales de la gestión ambiental:

Ciclo de Operación:

  1. Monitorear: La Eco Watch Station registra y transmite datos ambientales continuamente
  2. Responder: Si se detecta fuego, el Robot Bombero se activa automáticamente y combate el incendio
  3. Restaurar: Una vez controlado, el Robot Reforestador siembra semillas y riega el área afectada
  4. Verificar: La Eco Watch Station continúa monitoreando el terreno reforestado para evaluar su recuperación

Características Técnicas:

  • ⚡ Autonomía energética: Paneles solares y baterías recargables
  • 🎯 Precisión: Decisiones basadas en datos objetivos de sensores
  • 🔄 Eficiencia: Tiempo de respuesta reducido y reforestación optimizada
  • 📈 Escalabilidad: Replicable en otras regiones y ecosistemas
  • 💰 Accesibilidad: Materiales económicos y fáciles de conseguir
  • ♻️ Sostenibilidad: Uso de energías limpias y renovables

Código Arduino - Eco Watch Station

Sistema de monitoreo ambiental con sensores DHT22, humedad de suelo y detector de humo MQ-2.

eco_watch_station.ino 
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
#include "DHT.h"

// Configuración del DHT22
#define DHTPIN 9
#define DHTTYPE DHT22
DHT dht(DHTPIN, DHTTYPE);

// Pines para otros sensores
#define PIN_HUMEDAD_SUELO A0
#define PIN_MQ2 8  // salida digital del MQ-2

// Configuración del LCD
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  dht.begin();
  pinMode(PIN_MQ2, INPUT);
  
  lcd.init();
  lcd.backlight();
  lcd.clear();
  lcd.setCursor(0, 0);
  lcd.print("Sensores listos");
  delay(2000);
}

void loop() {
  // Leer temperatura y humedad ambiente
  float humedadAmbiente = dht.readHumidity();
  float temperatura = dht.readTemperature();
  
  if (isnan(humedadAmbiente) || isnan(temperatura)) {
    lcd.clear();
    lcd.setCursor(0, 0);
    lcd.print("Error DHT22");
    delay(2000);
    return;
  }
  
  // Leer humedad del suelo
  int valorHumedadSuelo = analogRead(PIN_HUMEDAD_SUELO);
  int humedadSuelo = map(valorHumedadSuelo, 1023, 0, 0, 100);
  if (humedadSuelo < 0) humedadSuelo = 0;
  if (humedadSuelo > 100) humedadSuelo = 100;
  
  // Leer MQ-2 (detección de humo)
  int mq2Digital = digitalRead(PIN_MQ2);
  String fuego = (mq2Digital == HIGH) ? "SI" : "NO";
  
  // Mostrar datos en LCD
  lcd.clear();
  lcd.setCursor(0, 0);
  lcd.print("T:");
  lcd.print(temperatura, 1);
  lcd.print("C H:");
  lcd.print(humedadAmbiente, 0);
  lcd.print("%");
  
  lcd.setCursor(0, 1);
  lcd.print("Suelo:");
  lcd.print(humedadSuelo);
  lcd.print("% F:");
  lcd.print(fuego);
  
  delay(2000);
}

Código Arduino - Robot Reforestador

Control de motores, servos para siembra y sistema de riego automatizado.

robot_reforestador.ino 
#include <Servo.h>

// Pines de motores
#define ENA 3
#define IN1 4
#define IN2 5
#define ENB 6
#define IN3 7
#define IN4 8

// Servos para siembra
#define SERVO1 9   // Brazo sembrador
#define SERVO2 10  // Abrir surco
#define SERVO3 11  // Tapar semilla

// Sistema de riego
#define BOMBA 12
#define VALVULA 13

Servo servo1, servo2, servo3;
char comando;

void setup() {
  Serial.begin(9600);
  
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(ENB, OUTPUT);
  pinMode(IN3, OUTPUT);
  pinMode(IN4, OUTPUT);
  pinMode(BOMBA, OUTPUT);
  pinMode(VALVULA, OUTPUT);
  
  servo1.attach(SERVO1);
  servo2.attach(SERVO2);
  servo3.attach(SERVO3);
  
  digitalWrite(BOMBA, LOW);
  digitalWrite(VALVULA, LOW);
}

void loop() {
  if (Serial.available() > 0) {
    comando = Serial.read();
    
    switch (comando) {
      case 'F':  // Adelante
        digitalWrite(IN1, HIGH);
        digitalWrite(IN2, LOW);
        digitalWrite(IN3, HIGH);
        digitalWrite(IN4, LOW);
        break;
      
      case 'B':  // Atrás
        digitalWrite(IN1, LOW);
        digitalWrite(IN2, HIGH);
        digitalWrite(IN3, LOW);
        digitalWrite(IN4, HIGH);
        break;
      
      case 'S':  // Stop
        digitalWrite(IN1, LOW);
        digitalWrite(IN2, LOW);
        digitalWrite(IN3, LOW);
        digitalWrite(IN4, LOW);
        break;
      
      case 'P':  // Proceso de siembra
        servo2.write(90); delay(500);  // Abrir surco
        servo1.write(45); delay(500);  // Soltar semilla
        servo3.write(90); delay(500);  // Tapar semilla
        servo1.write(0);
        servo2.write(0);
        servo3.write(0);
        break;
      
      case 'W':  // Activar riego
        digitalWrite(BOMBA, HIGH);
        digitalWrite(VALVULA, HIGH);
        delay(3000);  // Riega 3 segundos
        digitalWrite(BOMBA, LOW);
        digitalWrite(VALVULA, LOW);
        break;
    }
  }
}
